location 是最有用的 BOM 对象之一, 提供了当前窗口中加载文档的信息, 以及通常的导航功能。

这个对象独特的地方在于, 它既是 window 对象的属性, 也是 document 的属性, 也就是说, window.location 和 document.location 指向同一个对象。
